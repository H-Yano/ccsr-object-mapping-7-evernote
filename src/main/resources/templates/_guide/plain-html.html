シンプルなHTMLを書く

この画面は、最小限のスタイルを指定した Water.css を使用しています。
HTML要素を記述するだけで、このような画面を作成できます。
class属性は不要です。

オブジェクト ⇔ HTMLに戻る

ナビゲーション

トップに戻る | オブジェクト ⇔ HTML | オブジェクト ⇔ SQL | オブジェクト ⇔ JSON

ソースコード 
  
<header>
  <h2>ナビゲーション</h2>
  <nav>
    <a href="/">トップに戻る</a>
    | <a href="/guides/object-html">オブジェクト ⇔ HTML</a>
    | <a href="/guides/object-sql">オブジェクト ⇔ SQL</a>
    | <a href="/guides/object-json">オブジェクト ⇔ JSON</a>
  </nav>
</header>

文字の修飾

  • mark要素で囲むとハイライト表示ができる
  • strong要素で囲むと太字になる
  • em要素で囲むと斜体になる

ソースコード 
  
<ul>
  <li>mark要素で囲むと<mark>ハイライト表示ができる</mark></li>
  <li>strong要素で囲むと<strong>太字</strong>になる</li>
  <li>em要素で囲むと<em>斜体</em>になる</li>
</ul>

サマリーと詳細

詳細を閉じている

サマリー（クリックで詳細を表示／非表示）
details要素
summary要素と他の詳細情報の記述をまとめる
summary要素
サマリー情報
先頭の▽アイコンで開閉を表現する

詳細を開いている

サマリー（クリックで詳細を表示／非表示）
  • 詳細情報を開閉する
  • details要素にopen属性を指定すると、初期状態は詳細表示になる

ソースコード 
  
<h3>詳細を閉じている</h3>
  <details>
    <summary>サマリー（クリックで詳細を表示／非表示）</summary>
    <dl>
      <dt>details要素</dt>
      <dd>summary要素と他の詳細情報の記述をまとめる</dd>
      <dt>summary要素</dt>
      <dd>サマリー情報</dd>
      <dd>先頭の▽アイコンで開閉を表現する</dd>
    </dl>
  </details>
  <h3>詳細を開いている</h3>
  <details open>
    <summary>サマリー（クリックで詳細を表示／非表示）</summary>
    <ul>
      <li>詳細情報を開閉する</li>
      <li>details要素にopen属性を指定すると、初期状態は詳細表示になる</li>
    </ul>
  </details>
