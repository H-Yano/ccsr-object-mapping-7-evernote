アプリケーションの紹介

ハーブ栽培キットの販売会社が、商品を管理する社内アプリケーションです。
Thymeleaf-Springのドキュメント(英語) に登場するアプリケーションを元にしています。

このアプリケーションとドキュメントはオブジェクトと画面、オブジェクトとデータベース、オブジェクトとJSONデータをマッピングするやり方の学習用に作成しました。

使用している要素技術は以下の通りです。

  • 画面：Spring MVC (WebDataBinder), Thymeleaf
  • データベース：MyBatis SQL Mapper, H2 Database, Flyway
  • JSON(API)：Spring MVC (Jackson ObjectMapper), Swagger-UI
