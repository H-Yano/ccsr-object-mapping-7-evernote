オブジェクトマッピングの概要

ハーブ品種の品種番号と品種名を持つVarietyクラスを使って、 オブジェクトと画面、 オブジェクトとデータベース、 オブジェクトとJSONデータ のマッピングのやり方を説明します。 

  
package example.domain.model.kit.row;

/**
* ハーブの品種を表現するクラス
*/
public class Variety {
    VarietyNumber varietyNumber;
    String name = "";

    public String show() {
        return String.format("%s [%s]", name, varietyNumber);
    }
}
