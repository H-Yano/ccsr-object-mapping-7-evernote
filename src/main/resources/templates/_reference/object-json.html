オブジェクト ⇔ JSONのマッピング

ハーブの品種の一覧APIと登録APIを例に、オブジェクトとJSONとのマッピングを説明します。
マッピングの対象は、下記のVarietyクラスです。 

  
package example.domain.model.kit.row;

/**
* ハーブの品種を表現するクラス
*/
public class Variety {
    VarietyNumber varietyNumber;
    String name = "";

    public String show() {
        return String.format("%s [%s]", name, varietyNumber);
    }
}
